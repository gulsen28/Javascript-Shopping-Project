# JAVASCRIPT SHOPPING PROJECT

In this project, it is a Javascript and Jquery based project that provides a dynamically created product promotion and shopping. In its design, Bootstrap libary and Css are used together. It has navbar, slider, dropdown menu, card, form elements, alerts and footer in its structure. It is created by processing data from the service. The area created for the advertisement is fixed and is displayed until the display date. The general structure of the project,
1. Home
* Navbar

* Categories

* Slider

* Last added products


* footer

2. Product Detail


3. User Profile

In the category section, the hide feature with the toggle button has been used to be used in Accordion and on the mobile side. Except for the blog, all data is dynamically taken from the API source. A default image is assigned for product or advertisement images that may be empty from the API. Together with the product image control, products are taken according to the categoryID value. If there is no image, the default image is added.

In the add to cart section, if the user is not logged in, they are directed to the login page, and if there is no user information, to the registration page, and after logging in, the product to be added to the cart is returned.

After entering data in all fields in the form elements, it informs the user with an alert message. If no data is entered in all fields, a danger alert is given. All fields are designed as required. But against other possibilities, after checking all fields with regex, it informs the user with an alert message. Redirected to the login page.

User output is provided with LOG OUT on the profile screen. The user's data kept on the browser is removed with remove. Personal information is taken from the API and Update operations can also be performed. After the update, the user logs out and logs in again. All information is taken from the API source and transferred to value values.
